Χρήσιμα χαρακτηριστικά

  • Με όμορφη διεπαφή (UI) που προσαρμόζεται τέλεια στο τηλέφωνο ή ταμπλέτα σας
  • Περιλαμβάνει όλα τα φαρμακεία της Κύπρου μαζί με τις συντεταγμένες τους
  • Εμφάνιση των διημερεύοντων φαρμακείων (με συνεχή ενημέρωση)
  • Πατήστε στην διεύθυνση ενός φαρμακείου για να το δείτε στον χάρτη
  • Πατήστε στον αριθμό τηλεφώνου ενός φαρμακείου για να το καλέσετε
  • Αναζητήστε κοντινά φαρμακεία
  • Μοιραστείτε εύκολα τις πληροφορίες για ενα συγκεκριμένο φαρμακείο μέσω email, SMS, κλπ
  • Πλήρης υποστήριξη για Ελληνικά και Αγγλικά (και για όσους προτιμούν, και Γαλλικά και Γερμανικά)
  • Ενσωματώνει μηχανισμό αναζήτησης για απευθείας εύρεση φαρμακείων (δεστε τις εικόνες)

Πολιτική προστασίας προσωπικών δεδομένων

Ο Οδηγός Φαρμακείων Κύπρου λαμβάνει πολύ σοβαρά την προστασία των προσωπικών σας δεδομένων!

Οι μόνες απαιτήσεις της εφαρμογής αφορούν χρήση του internet, η οποία απαιτείται για να συγχρονίζει τις πληροφορίες που αφορούν τα διημερεύοντα φαρμακεία, και γεωγραφική θέση, που χρησιμοποιείται για τον υπολογισμό των κοντινών σας φαρμακείων. Παρακαλώ σημειώστε ότι η κατά προσέγγιση τοποθεσία σας λαμβάνεται ανώνυμα κατά την διάρκεια αναζήτησης κοντινών φαρμακείων, για σκοπούς έρευνας.

Επίσης χρησιμοποιούμε το Google Analytics για να παρακολουθούμε την χρήση της εφαρμογής, και το AdMob για να εμφανίζουμε διαφημίσεις.
